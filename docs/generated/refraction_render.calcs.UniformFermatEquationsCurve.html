refraction_render.calcs.UniformFermatEquationsCurve¶

class refraction_render.calcs.UniformFermatEquationsCurve(R0, n, dndy, args=())[source]¶

Solver for light ray in a 2D Polar geometry with \(n=n(y)\).

This object takes in three user defined functions: \(n(y), \frac{\partial n(y)}{\partial y}\) and uses these functions to solve Fermat’s equations for the path of a light ray.
