Calcs module (refraction_render.calcs)¶

classes for solving Fermat’s principle:¶

FermatEquationsEuclid(n, dndx, dndy[, args]) Solver for light ray in a 2D Euclidian geometry for \(n=n(x,y)\).
FermatEquationsCurve(R0, n, dnds, dndy[, args]) Solver for light ray in a 2D Polar geometry for \(n=n(s,y)\).
UniformFermatEquationsEuclid(n, dndy[, args]) Solver for light ray in a 2D Euclidian geometry with \(n=n(y)\).
UniformFermatEquationsCurve(R0, n, dndy[, args]) Solver for light ray in a 2D Polar geometry with \(n=n(y)\).

classes for solving rays in atmospheric model:¶

Calc(FermatEquations_obj) Base class for Calculators.
CurveCalc([R0]) Calculator used for calculating rays on a spherical earth in an atmosphere.
CurveNoRefraction([R0]) Calculator used for calculating rays on a sphere earth in no atmosphere.
FlatCalc(**std_atmosphere_args) Calculator used for calculating rays on a flat earth in an atmosphere.
FlatNoRefraction() Calculator used for calculating rays on a flat earth in no atmosphere.

class for solving standard atmosphere¶

std_atmosphere([h0, T0, P0, g, dT, …]) Object which calculates the standard atmospheric model.
